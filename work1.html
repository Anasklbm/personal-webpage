Easy shop

This project is a web based shopping system for existing shops. The project objective is to deliver the online shopping application into web platform. This project is an attempt to provide the advantages of online shopping to customers of a real shop. It helps buying the products in the shop anywhere through internet. Thus the customer will get the service of online shopping and home delivery from his favourite shop. This system can be implemented to any shop in the locality or to multinational branded shops having retail outlet chains.

The Challenge

In the proposed system customer need not go to the shop for buying the products. He can order the product he wishes to buy through the application. Shop owner can appoint moderators who will help owner in managing the customers and product orders. The system also recommends a home delivery system for the purchased products

The Solution

The purpose of the system is to develop a web portal for purchasing items from shops effectively and efficiently compared to the existing system. The system is developed in considering all issues related to all user which are included in this system. It helps the users in making order easily. The proposed system would attract customers and adds to the efficiency of maintaining the shops and billing sections. Thus we can conclude that the web portal is easy to implement and very customizable according to the user requirement
